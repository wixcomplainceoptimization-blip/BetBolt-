# ⚡ BetBolt - AI Betting Assistant Bot

BetBolt is a Telegram bot that provides real-time betting odds analysis, predictions, and insights for various sports leagues.

## 🚀 Features

- 📊 **Live Odds**: Get real-time odds from multiple bookmakers
- 🔮 **Smart Predictions**: AI-powered analysis of match outcomes
- 📈 **Confidence Scoring**: Each prediction comes with a confidence level
- 🌍 **Multiple Leagues**: Support for Premier League, La Liga, Serie A, and more
- ⚡ **Instant Response**: Fast and reliable performance

## 📋 Prerequisites

- Python 3.8+
- Telegram Bot Token (from @BotFather)
- The Odds API Key (from https://the-odds-api.com/)
- GitHub Account
- Railway Account
